## مقدمة

**سكربت ريكس** هو إضافة بلغة PHP تتيح لك بناء مواقع إلكترونية متعددة اللغات بسهولة ومرونة، وتأتي الآن مع خدمة قوية لترجمة الملفات تلقائياً عبر واجهة الموقع الخاصة (File Translator).

---

## موقع مترجم الملفات (File Translator)

وفرنا لك موقع "مترجم الملفات" الذي يتيح ترجمة ملفات اللغة بصيغة JSON لأي لغة تريدها بسهولة، أسرع وبجودة عالية:

### ميزات الموقع:

- **دعم الترجمة التلقائية:** ترجمة ملفات JSON كاملة لأي لغة بنقرة واحدة.
- **واجهة سهلة وبسيطة:** يمكنك رفع الملف وتحميل الترجمة بسرعة وبساطة.
- **توليد Token شخصي:** أمان عالٍ باستخدام التوكن الخاص لكل مستخدم.
- **API لدمج الترجمة:** اربط النظام مع مشاريعك أو سير عملك البرمجي مباشرة.
- **تعدد اللغات المدعومة:** يدعم الترجمة لمعظم لغات العالم بدون تعقيد.
- **تاريخ الترجمة والإدارة:** بإمكانك العودة لترجماتك السابقة وتحميلها متى شئت.
- **استخدام مجاني أو بخطط تناسب الجميع.**

### خطوات استخدام الموقع

#### 1. إنشاء حساب جديد

- توجه إلى [منصة المترجم](https://abbasawad25.epizy.com/abbasawad26) واضغط على "إنشاء حساب".
- فعّل بريدك الإلكتروني، الآن لديك حساب جاهز.

#### 2. توليد مفتاح التوكن (API Token)

- بعد تسجيل الدخول، ستجد في لوحة التحكم خيار **توليد Token جديد**.
- احتفظ بالـ Token بشكل سري، ستستخدمه للربط مع API الترجمة.

#### 3. ترجمة ملف JSON عبر الموقع

- ادخل إلى قسم "ترجمة ملف".
- اختر اللغة المستهدفة.
- قم برفع ملف اللغة (json).
- سيتم ترجمة الملف في لحظات ويمكنك تحميل الملف المترجم مباشرة.

#### 4. ربط الـ API واستعماله برمجياً

- استخدم الـ Token في الهيدر `Authorization` في واجهة API للموقع.
- أرسل ملف JSON المطلوب الترجمة إليه، وحدد اللغة المطلوبة.
- مثال على استخدام API ببرمجية PHP (مثال توضيحي):

    ```php
    $url = "https://abbasawad25.epizy.com/abbasawad26/api/translate";
    $data = [
      "language" => "tr", // اللغة المستهدفة
      "file" => base64_encode(file_get_contents("ar.json")),
    ];
    $opts = [
      "http" => [
        "method" => "POST",
        "header" => "Authorization: Bearer YOUR_TOKEN_HERE\r\nContent-Type: application/json\r\n",
        "content" => json_encode($data)
      ]
    ];
    $context = stream_context_create($opts);
    $result = file_get_contents($url, false, $context);
    file_put_contents("tr.json", base64_decode(json_decode($result, true)['file']));
    ```

- **الإجابة تكون بملف JSON مترجم جاهز للعمل في مشروعك!**

## سكربت ريكس — الميزات الإضافية

- **إدارة اللغات:** إضافة، تحديث، تعديل، حذف، تصدير واستيراد لأي لغة.
- **دعم صيغة JSON:** استيراد وتصدير ملفات اللغات كـ JSON.
- **سهولة الاستخدام:** عملية التثبيت والتكامل بسيطة — فقط أضف مجلد `languages` لمشروعك واستورد ملف الإعدادات.
- **ترجمة مباشرة:** يمكنك ترجمة الملفات من خلال لوحة التحكم مباشرة.
- **دعم API:** جميع مزايا الإضافة متاحة عبر واجهة برمجية (API) مرفقة.
- **استيراد/تصدير اللغات:** سهولة نقل اللغات بين المشاريع أو الأجهزة.

## التثبيت والاستخدام

1. **اسحب ملفات السكربت من مجلد tools إلى مشروعك.**
2. **أضف مجلد languages** إلى موقعك.
3. **استدعِ ملف إعدادات اللغات** في رأس الصفحات المطلوبة:

    ```php
    <?php include("languages/langConfig.php"); ?>
    ```

4. **تغيير/إضافة لغة** يتم بإضافة شرط بسيط:

    ```php
    else if ($_GET['lang'] == "en") {
        $_SESSION['lang'] = "en";
    }
    // يمكنك تغيير اللغة الافتراضية من العربية لأي لغة أخرى
    ```

5. **إضافة لغة جديدة**
    - انسخ ملف `ar.php`، أعد تسميته مثلاً إلى `en.php` أو `tr.php` حسب رغبتك.
    - عدل المحتوى إلى ترجمة النصوص المطلوبة.

6. **استخدام النصوص المترجمة داخل الصفحات:**

    ```php
    <?php echo $lang['login']; ?>
    ```

    لإضافة نص جديد، افتح ملف اللغة المناسب (مثلاً ar.php) وأضف:
    ```php
    'name' => ['name'],
    ```

7. **استيراد/تصدير ملف لغة**
    - تصدير اللغة يتم كملف JSON.
    - يمكنك أيضاً استبدال ملف لغة بآخر بسهولة (ميزة هامة عند التحديث التلقائي).

## أمثلة على الاستخدام

```php
<?php include("languages/langConfig.php"); ?>
<h1><?php echo $lang['welcome']; ?></h1>
<a href="?lang=en">English</a> | <a href="?lang=ar">العربية</a>
```
> سيعرض العنوان بلغة المستخدم تلقائياً حسب اختياره.
